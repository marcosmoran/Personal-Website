Unreleased Kinect Application

Technologies used: Unity
Credits: Antiloop

Created for a large dog food company who is planning an immersive marketing event, the application will take in a subject, and using Kinect camera and Unity, the subject will then be shown on a screen applying a variety of eye-catching effects on their point cloud image. The application was split into several parts, development of the application logic and user interface and the special effect applied to the subject, which was made using Unity’s special effect graph.

Torreciudad 360 Experience

Technologies used: Unity, Virtual Reality
Credits: Antiloop, Playmedia

Made for a large museum in Spain, Torreciudad 360 is a VR experience that takes the viewer back in time on an educational journey. The application was built on Unity and uses Pico virtual reality headsets, the application works in two parts, a user interface on a touch screen device where users select their desired languages, which then communicates to the Pico headset with the use of OSC to play the correct experience.

SoundScape

Technologies used: Java, FFT

SoundScape was my final year project at university, it involved a 2D game that would react visually to user-inputted music, creating a hybrid of a game and visualizer. I had the chance to showcase my game to many members of the industry at CompShow 2018 and won runner-up for best overall project. The project was made using Java's creative coding library Processing and used Fast Fourier Transforms to analyze the audio signal. I am currently remaking this using Three.js.

CompShow 2018

CompShow is an annual event held at the University of Hertfordshire, where a selection of Computer Science students are chosen to showcase their final year projects to members of the industry. I was presented with the Gordon Brand Prize after achieving the highest overall grade in the module User Experience, and won runner-up for best overall final year project, for my audio reactive game “SoundScape”.
